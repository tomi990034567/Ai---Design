Designing design methods

(ChatGPT)
Visit GitHub Page

ChatGPT is an AI assistant created by Anthropic to be helpful, harmless, and honest. It can provide information to users by processing natural language. Some designers have started using ChatGPT to help explore different design methods and approaches. By having a conversation with ChatGPT, designers can get feedback on early design concepts or brainstorm new directions for a project. The AI is knowledgeable about general design principles and practices across industries. It can recommend methods that may help solve particular design challenges. Some find ChatGPT useful for idea generation in the initial phases before committing to specific solutions. It allows designers to rapidly iterate conversations and receive responses without having to research information themselves. However, as an AI, ChatGPT's responses are limited since it does not truly understand contexts in the same way humans do. The best designs usually emerge from real collaboration between people. But ChatGPT can still serve as a helpful early stage tool to supplement traditional design processes.

The Results

I have been working for E.ON in the learning and development department. Recently, I was tasked with designing a new learning journey on Smart Grids and decided to use ChatGPT to help with some of the content creation. It provided the following outline and description for the learning journey, which I think captures the key elements we want to convey: It introduces what a Smart Grid is and its role in the energy transition. The learning journey aims to provide learners with the fundamentals of Smart Grids and show why modernizing energy networks is necessary. It also gives insight into concrete Smart Grid projects E.ON is working on. Interactive scenarios are designed to illustrate the construction and operation of a Smart Grid in a hands-on way. Learners can observe E.ON experts in action and assist them with decision making. This allows learning how new technologies like drones improve network efficiency. By the end, learners should understand Smart Grid basics and know about E.ON's projects. Additional content will be added to provide deeper insights. Both guided and self-paced options are offered for completing the journey. Overall, I think ChatGPT did a good job of capturing the key elements we want to convey about Smart Grids and E.ON's role in a concise yet informative manner. The outline provides a solid foundation to start developing the full learning journey content:
